# query-leyu-notes

本仓库用于归档和发布多个独立 HTML 页面，主要用于记录和展示与“乐娱”相关的各类信息、备注及页面内容。仓库内容不针对任何单一域名或特定网站，仅作为页面资源的集合与备份。

## 项目简介

`query-leyu-notes` 是一个静态 HTML 页面归档仓库，旨在以清晰、简洁的方式整理和发布与“乐娱”主题相关的页面。每个页面均为独立 HTML 文件，便于直接浏览或通过 GitHub Pages 等静态托管服务访问。

仓库内容可能包含说明、记录、参考信息等，不涉及任何商业推广或诱导行为。

## 目录说明

```
/
├── index.html          # 仓库首页或导航页
├── pages/              # 存放具体 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 页面所需的静态资源（如样式、图片等）
│   ├── style.css
│   └── images/
└── README.md           # 本文件
```

- `index.html`：可选，作为仓库的入口页面，提供页面列表或导航。
- `pages/`：所有独立的 HTML 页面均存放于此目录，文件名应具有描述性。
- `assets/`：存放页面依赖的 CSS、JavaScript、图片等资源文件。

## 页面归档说明

- 每个 HTML 页面均为独立文件，可直接在浏览器中打开。
- 页面内容可能包含文本、表格、链接、样式等，但不会包含恶意代码或诱导性内容。
- 页面之间无强制依赖关系，可单独使用或组合浏览。
- 归档内容可能随时间更新，历史版本可通过 Git 提交记录查看。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接打开 `pages/` 目录下的 HTML 文件即可浏览。
3. 若通过 GitHub Pages 部署，可访问对应 URL 在线查看。
